# Scope 1/2/3 排放定义与化工行业实操

## 基本定义（GHG Protocol）

| 范围 | 含义 | 化工企业典型来源 |
|------|------|---------------|
| Scope 1 | 企业自有/控制设施的**直接**排放 | 锅炉/工艺炉燃料燃烧；电石/合成氨/纯碱等工艺过程 CO2；甲烷逸散；含氟气体 |
| Scope 2 | 外购电力/热力/蒸汽对应的**间接**排放 | 外购电网电力；外购热网蒸汽 |
| Scope 3 | 价值链上下游**间接**排放（共 15 类） | 上游煤炭/石脑油运输；下游产品运输；原料生产隐含碳；员工差旅 |

## Scope 1 在化工行业的细分

化工行业 Scope 1 通常拆为 4 个口径：

1. **燃料燃烧**：锅炉、自备电厂、工艺加热炉煤/气/油燃烧
2. **工艺过程**：
   - 电石法 PVC：约 1.7-2.0 t CO2/t PVC（电石生成 + 氯乙烯单体合成）
   - 合成氨：约 1.5-2.0 t CO2/t NH3（蒸汽转化制氢释放 CO2）
   - 煤制烯烃：约 8-12 t CO2/t 烯烃（煤气化高强度）
   - 煤制天然气（SNG）：约 6-7 t CO2/千 m3 SNG
3. **甲烷逸散**：油气开采/储运逸散，化工企业占比通常较小
4. **含氟气体**：制冷设备、消防、半导体级气体生产

## Scope 2 在地区A电网的特殊性

地区A电网（属西北电网）**单位发电碳强度高**，原因是煤电占比高。

| 电网区域 | 平均排放因子（t CO2/MWh） | 备注 |
|----------|--------------------|------|
| 西北区域电网（含地区A） | ~0.86 | 煤电占比 75%+ |
| 华北区域电网 | ~0.78 |  |
| 华东区域电网 | ~0.62 | 核电、气电较多 |
| 南方电网 | ~0.51 | 水电、核电较多 |

> **影响**：同样 1 亿 kWh 外购电，地区A化工企业 Scope 2 排放比东部企业高约 70%。这是地区A化工企业绿电消纳率被高度关注的原因。

## Scope 3 化工企业的常见误区

- 多数中小化工企业**只报 Scope 1+2**，跨国客户 / 出口欧盟（CBAM）会要求 Scope 3
- Scope 3 中"原料生产隐含碳"对煤化工企业最重要，因为煤气化是高碳工艺，但煤本身的开采运输不计入 Scope 1
- 员工差旅、办公能耗等占比通常 < 1%，但合规报告仍需列示

## CBAM 与 Scope 数据要求

欧盟碳边境调节机制（CBAM）目前覆盖钢铁、铝、化肥、水泥、电力、氢，**化肥行业是化工行业最直接受影响的子行业**：

- 申报数据：单位产品**直接 + 间接排放强度**（即 Scope 1 + Scope 2）
- 核算口径：欧盟 EU ETS 兼容口径
- 第三方核查：欧盟认可的核查机构

地区A化肥企业（化工企业E、化工企业G等）若有出口欧盟的产品，需提前准备 CBAM 申报数据。

## 实操建议

1. **MRV 数据底盘**：尽早搭建 Scope 1+2 月度台账，避免年度核查时手忙脚乱
2. **绿电采购**：通过绿证或直购可降低 Scope 2，对 CBAM 申报有显著利好
3. **CCER 抵消**：余热发电、合成氨节能、可再生能源消纳等可开发 CCER，最高抵消 5% 配额履约
4. **工艺替代**：电石法 PVC 转乙烯法（需消耗石脑油/乙烷），降本但需评估总拥有成本
